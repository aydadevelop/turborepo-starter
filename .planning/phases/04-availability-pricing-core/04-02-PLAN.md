Quote calculation formula:
```
durationMinutes = (endsAt - startsAt) in minutes
durationHours = durationMinutes / 60  (non-integer ok for calculation)
baseCents = round(profile.baseHourlyPriceCents * durationHours)
adjustmentCents = apply resolved rules (percent → round(baseCents * value/100); flat_cents → value)
subtotal = baseCents + adjustmentCents
serviceFeeCents = round(subtotal * profile.serviceFeeBps / 10000)
taxCents = round((subtotal + serviceFeeCents) * profile.taxBps / 10000)
totalCents = subtotal + serviceFeeCents + taxCents
```
